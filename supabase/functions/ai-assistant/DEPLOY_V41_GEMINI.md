# V41.0 — Free Brain Swap (Google Gemini, δωρεάν)

Αυτή η έκδοση του edge function `ai-assistant` προσθέτει **δωρεάν** AI μέσω
Google Gemini (`gemini-2.0-flash`), χωρίς καμία αλλαγή στην εφαρμογή (app.js / index.html / styles.css).

## Γιατί δεν επηρεάζεται η εφαρμογή στα κινητά
- Αλλάζει **μόνο** το αρχείο του edge function στο Supabase.
- Το input payload και η απάντηση `{ ok, answer }` μένουν **ακριβώς ίδια**.
- Αν δεν υπάρχει κλειδί, ή το μοντέλο αποτύχει, απαντά **ο ίδιος τοπικός βοηθός όπως σήμερα**.
- Καμία κλήση στο μοντέλο δεν ρίχνει 500 — στη χειρότερη περίπτωση γυρνά τοπική απάντηση.

## Σειρά προτεραιότητας απάντησης
1. `GEMINI_API_KEY` υπάρχει  → Google Gemini (δωρεάν)
2. αλλιώς `OPENAI_API_KEY` υπάρχει → OpenAI (όπως πριν)
3. αλλιώς → τοπική λογική (localFallback)

## Βήματα ενεργοποίησης (μία φορά)

1. Πάρε δωρεάν κλειδί από το Google AI Studio: https://aistudio.google.com/apikey

2. Πρόσθεσέ το ως secret στο Supabase (δεν μπαίνει ΠΟΤΕ στον client):
   ```bash
   supabase secrets set GEMINI_API_KEY=το_κλειδι_σου
   ```
   (Προαιρετικά: `supabase secrets set GEMINI_MODEL=gemini-2.0-flash`)

3. Κάνε deploy μόνο αυτή τη συνάρτηση:
   ```bash
   supabase functions deploy ai-assistant
   ```

## Δοκιμή πριν το εμπιστευτείς
- Άνοιξε την εφαρμογή, πήγαινε στον AI Βοηθό και ρώτησε κάτι.
- Αν η απάντηση είναι πιο «φυσική»/αναλυτική → δουλεύει το Gemini.
- Αν για οποιονδήποτε λόγο δεν απαντήσει το cloud, θα δεις την τοπική απάντηση
  ακριβώς όπως σήμερα (μηδενική διαφορά για τους συναδέλφους).

## Αναίρεση (rollback)
Αν θέλεις να γυρίσεις πίσω, απλώς ξανα-κάνε deploy την προηγούμενη έκδοση του αρχείου,
ή σβήσε το secret:
```bash
supabase secrets unset GEMINI_API_KEY
```
Χωρίς κλειδί, η συμπεριφορά είναι πανομοιότυπη με την παλιά (OpenAI ή τοπικό).

## Δωρεάν όρια
Το Gemini Flash έχει γενναιόδωρο δωρεάν tier (όρια ανά λεπτό/ημέρα). Για ένα γραφείο
τελετών με λίγες ερωτήσεις την ημέρα, μένεις άνετα μέσα στο δωρεάν.

## Σημείωση ιδιωτικότητας (GDPR)
Τα δεδομένα τελετών είναι ευαίσθητα. Το payload που φεύγει είναι ήδη «συμπιεσμένο»
(compactPayloadForPrompt). Σε επόμενη έκδοση (V41.1) προτείνεται ψευδωνυμοποίηση
ονομάτων πριν την αποστολή.
