# Η Μέρα Μου

Νέο, ανεξάρτητο project μέσα σε αυτό το repo — δεν έχει καμία σχέση με το
υπάρχον funeral-card SaaS (`saas/`, `karta grafeio/`).

- Design doc / product vision: [`docs/design.md`](docs/design.md)
- Πρωτότυπο Φάσης Α (static PWA, χωρίς build step): [`app/`](app)

## Τοπική εκτέλεση του πρωτοτύπου

```
cd mera-mou/app
python3 -m http.server 8080
```

Άνοιξε `http://localhost:8080` στο κινητό ή στο desktop browser.
